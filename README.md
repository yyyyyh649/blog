# 随想录 - 个人博客

一个简约、舒适、高级风格的个人博客，用于记录人生感悟、年度计划和日常随想。

## 预览

这是一个静态博客网站，可以直接通过 GitHub Pages 部署。

🔗 **在线访问**: [https://yyyyyh649.github.io/blog/](https://yyyyyh649.github.io/blog/)

## 特性

- 🎨 简约高级的视觉设计
- 📱 完全响应式，支持移动端
- 🌓 舒适的配色方案
- ✍️ 优雅的中文排版
- 🚀 纯静态，加载快速

## 内容分类

- **人生感悟** - 关于成长、选择、人生的思考
- **年度计划** - 新年愿望、目标规划、阶段性复盘
- **随想杂记** - 日常生活中的所见所闻所感

## 文件结构

```
blog/
├── index.html          # 首页
├── posts.html          # 文章列表页
├── about.html          # 关于我页面
├── css/
│   └── style.css       # 样式文件
└── posts/
    ├── reflection-on-growth.html   # 示例文章：关于成长
    ├── 2025-plans.html             # 示例文章：年度计划
    └── daily-thoughts.html         # 示例文章：日常随想
```

## 部署

### GitHub Pages

1. 进入仓库设置 (Settings)
2. 找到 Pages 选项
3. 在 Source 中选择 `main` 分支
4. 保存后等待部署完成

### 本地预览

直接用浏览器打开 `index.html` 文件即可预览。

## 添加新文章

1. 在 `posts/` 目录下创建新的 HTML 文件
2. 参考现有文章的结构编写内容
3. 在 `index.html` 和 `posts.html` 中添加文章链接

## 自定义

- 修改 `css/style.css` 中的 CSS 变量可以调整颜色主题
- 修改 `about.html` 添加个人信息
- 替换文章内容为自己的文字
